# B10.0

## Changes

- Added Create Connected
- Added Straining recipe for Tiny Chaos Fragments
- Banned Submerged Nether Star block recipes now use Extended Craftings one instead, craft to convert
- Added Fluid and Chemical Quantum Compressor
- Removed all singularites
- Added 9 new singulaities 
- Custom Portal now support create trains if you wanna use it 
- Changed Ultimate Singularity recipe
- Added The Ultimate Ingot Recipe
- Finished final end game questlines
- Fully added CCV quests and rewards
- Infinity Chapter now has rewards and is now final
- Sextuple Compressed Cobblestone can now be replicated
- Tweaked Draconium Core recipe

## Fixes

- Fixed Prismarine and Certus Ingot having the incorrect ingot tag
- Missing Water and Air Purifier recipes
- Heat Core recipe in Forcecraft
- Empty PCB crafting being odd

# Known Issues

- Fluid and Chemical Compressor textures not final 
- Missing textures from Submerged
