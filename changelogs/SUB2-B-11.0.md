# B11.0

## Changes

- Updated to NeoForge 21.1.219
- Dreaded Draconic Compound is no longer consumed in recipes
- Dreaded Draconic Compound recipe is now slightly more expensive to adjust for above change
- Updated Gold Leaf quest information
- Updated baykok quest information
- Added Liquid Fissile Fuel, used for casting made from Fissel Fuel, used 1mb per craft and crafts at 10 ticks per item
- Reduced crafting times in Dragon Forges 
- Recuced Strainer straining time from 220 to 160 ticks
- Added more wildlife to the sea mostly hostile
- Protecting armor now takes damage after 2 seconds whilst in water
- Orichalcum Blend now uses ingots instead of raw ores 
- Added Suspicious Seed recipe
- Added new secret quest
- Changed trial spawner loot to include crab claws

## Fixes

- Assembler recipe for Dreaded Draconic Compound now uses Ice Dragon Heart
- Removed Crushing Wheels recipe that use deepslate ore 
- Ultimate Singularity recipe
- Fixed dupe issue with star forge

## Known Issues

- Missing textures from Atlantis
